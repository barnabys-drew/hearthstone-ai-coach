# Security and privacy policy

This repository handles user-provided Hearthstone collection data. Treat that data as private.

## Please do not share publicly

- real `collection.json` or CSV exports,
- HSReplay `account_lo` URLs,
- browser Cookie headers,
- Battle.net or tracker account IDs,
- screenshots of logged-in account pages.

## Reporting a vulnerability

If you find a privacy or security issue, please open a private GitHub security advisory if available, or contact the maintainer directly. Do not include real collection exports or cookies in public issues.

## Scope

The scripts are local, dependency-free Python tools. They do not log in to Battle.net, automate the Hearthstone client, or attempt to bypass any access controls.
